About Me

Hi! My name is Kane York. I and my brother both often go by the name of Riking. He's more into games than I am, so if you see a Riking in a game, it's probably him. I frequent irc.esper.net and sometimes offer help to people making Minecraft mods.

I've had experience with programming Java, Python, Lua, and Windows Batch (still hate it).

My first-ever standalone program was "McModChooser", written in Windows Batch script. The code is now hosted at https://github.com/riking/mcmodchooser.
